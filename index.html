Windows

Microsoft Windows, powszechnie określany jako Windows, to grupa kilku zastrzeżonych rodzin graficznych systemów operacyjnych, z których wszystkie są opracowywane i sprzedawane przez firmę Microsoft. Każda rodzina obsługuje określony sektor przemysłu komputerowego. Aktywne rodziny Microsoft Windows obejmują Windows NT i Windows IoT; mogą one obejmować podrodziny (np. Windows Server lub Windows Embedded Compact) (Windows CE). Nieistniejące rodziny Microsoft Windows obejmują Windows 9x, Windows Mobile i Windows Phone.

Jak powstał system Windows?

Firma Microsoft wprowadziła środowisko operacyjne o nazwie Windows 20 listopada 1985 r. jako graficzną powłokę systemu operacyjnego dla MS-DOS w odpowiedzi na rosnące zainteresowanie graficznymi interfejsami użytkownika (GUI). Microsoft Windows zdominował światowy rynek komputerów osobistych (PC) z ponad 90% udziałem w rynku, wyprzedzając Mac OS, który został wprowadzony w 1984 roku. Apple zaczął postrzegać system Windows jako nieuczciwą ingerencję w ich innowacje w zakresie opracowywania GUI wdrożonych w produktach takich jak Lisa i Macintosh (ostatecznie rozstrzygnięte w sądzie na korzyść Microsoftu w 1993 r.). Na komputerach PC Windows jest nadal najpopularniejszym systemem operacyjnym we wszystkich krajach. Jednak w 2014 r. Microsoft przyznał się do utraty większości całego rynku systemów operacyjnych na rzecz Androida z powodu ogromnego wzrostu sprzedaży smartfonów z Androidem. W 2014 r. liczba sprzedanych urządzeń z systemem Windows była mniejsza niż 25% liczby sprzedanych urządzeń z systemem Android. To porównanie może jednak nie być w pełni trafne, ponieważ oba systemy operacyjne tradycyjnie są skierowane do różnych platform. Mimo to liczby dotyczące korzystania z systemu Windows przez serwery (porównywalne z konkurencją) pokazują jedną trzecią udziału w rynku, podobnie jak w przypadku użytkowników końcowych. Według stanu na październik 2021 r. najnowsza wersja systemu Windows na komputery PC i tablety to Windows 11 w wersji 21H2. Najnowsza wersja dla urządzeń wbudowanych to Windows 11, wersja 21H2. Najnowsza wersja dla serwerów to Windows Server 2022, wersja 21H2. Specjalistyczna wersja systemu Windows działa również na konsolach do gier Xbox One i Xbox Series X/S.

Genealogia

Microsoft, twórca systemu Windows, zarejestrował kilka znaków towarowych, z których każdy oznacza rodzinę systemów operacyjnych Windows skierowanych do określonego sektora przemysłu komputerowego. Od 2014 r. aktywnie rozwijano następujące rodziny systemu Windows:

Windows NT: Rozpoczęty jako rodzina systemów operacyjnych z Windows NT 3.1, systemem operacyjnym dla serwerów i stacji roboczych. Obecnie składa się z trzech podrodzin systemów operacyjnych, które są wydawane niemal w tym samym czasie i mają to samo jądro:

Windows: system operacyjny dla popularnych komputerów osobistych i tabletów. Najnowsza wersja to Windows 11. Głównym konkurentem tej rodziny jest macOS firmy Apple dla komputerów osobistych oraz iPadOS i Android dla tabletów (por. udział w wykorzystaniu systemów operacyjnych § udział w rynku według kategorii).

Windows Server: system operacyjny dla komputerów serwerów. Najnowsza wersja to Windows Server 2022. W przeciwieństwie do swojego brata klienta, przyjęła silny schemat nazewnictwa. Głównym konkurentem tej rodziny jest Linux. (por. udział w wykorzystaniu systemów operacyjnych § udział w rynku według kategorii)

Windows PE: uproszczona wersja swojego rodzeństwa Windows, przeznaczona do działania jako działający system operacyjny, używana do instalowania systemu Windows na komputerach typu „bare-metal” (zwłaszcza na wielu komputerach jednocześnie), odzyskiwania lub rozwiązywania problemów. Najnowsza wersja to Windows PE 10.

Windows IoT (wcześniej Windows Embedded): Początkowo firma Microsoft opracowała Windows CE jako system operacyjny ogólnego przeznaczenia dla każdego urządzenia, które miało zbyt ograniczone zasoby, aby można je było nazwać pełnoprawnym komputerem. Ostatecznie jednak nazwa systemu Windows CE została zmieniona na Windows Embedded Compact i została umieszczona pod znakiem towarowym Windows Compact, na który składają się również Windows Embedded Industry, Windows Embedded Professional, Windows Embedded Standard, Windows Embedded Handheld i Windows Embedded Automotive.

następujące rodziny Windows nie są już rozwijane:

Windows 9x: system operacyjny skierowany na rynek konsumencki. Przerwane z powodu nieoptymalnej wydajności. (PC World nazwał swoją ostatnią wersję Windows Me, jednym z najgorszych produktów wszechczasów.) Microsoft obsługuje teraz rynek konsumencki za pomocą systemu Windows NT.

Windows Mobile: poprzednik Windows Phone, był to system operacyjny telefonu komórkowego. Pierwsza wersja nazywała się Pocket PC 2000; trzecia wersja, Windows Mobile 2003, jest pierwszą wersją, w której zastosowano znak towarowy Windows Mobile. Ostatnia wersja to Windows Mobile 6.5.

Windows Phone: system operacyjny sprzedawany tylko producentom smartfonów. Pierwszą wersją był Windows Phone 7, a następnie Windows Phone 8 i Windows Phone 8.1. Jego następcą został Windows 10 Mobile, który również został wycofany.
